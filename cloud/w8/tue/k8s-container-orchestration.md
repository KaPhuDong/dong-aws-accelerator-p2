# W8 Day B - K8s Container/Orchestration

## Mục tiêu

- Ôn nền tảng container, image và runtime.
- Hiểu các object Kubernetes cơ bản: Pod, Deployment, ReplicaSet, Service.
- Nắm probes, ConfigMap, Secret và NetworkPolicy ở mức foundation.
- Chuẩn bị môi trường Docker Desktop, minikube và kubectl trước onsite.

## Checklist

- [ ] Cài Docker Desktop hoặc Docker Engine.
- [ ] Cài kubectl.
- [ ] Cài minikube.
- [ ] Chạy `kubectl version --client`.
- [ ] Chạy `minikube start`.
- [ ] Deploy thử một workload đơn giản.
